## Deploy to GitHub Pages
1. Create a GitHub repo
2. Upload all files (drag & drop or `git push`)
3. Settings → Pages → Source: main / root
4. Live at `https://yourusername.github.io/repo-name/`

## Images
All images are loaded directly from `theasianbanker.com` via CDN — no local assets needed.
`onerror` fallbacks handle any broken images gracefully.
